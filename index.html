KidneyCare System

Full Stack Development & System Architecture

Overview

KidneyCare is a specialized health tracking mini-app built for LINE OA. It allows patients to record daily vital signs, water intake, and interdialytic weight gain (IDWG). The system calculates health statuses in real-time, logs data securely to Google Sheets, and sends automated push notifications back to the user via LINE.

The Challenge

Initially built as a standard Google Apps Script Web App, the system faced severe CORS and timeout issues. Modern mobile browsers and LINE's In-App Browser block third-party cookies and restrict scripts within GAS's mandatory <iframe>, causing the LINE LIFF SDK initialization to fail silently.

The Solution

Implemented a Decoupled Architecture. I migrated the frontend to GitHub Pages to completely bypass Google's iframe wrapper. Google Apps Script was refactored purely into a REST API (using doPost), handling backend logic and LINE Push Messaging securely without iframe interference.
